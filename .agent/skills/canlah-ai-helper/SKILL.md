# CANLAH AI Agent Skill

This skill enables agents inside the Antigravity IDE to understand, leverage, and interface with the **CANLAH AI** ecosystem. It provides the structured knowledge and prompts required to interact with the 10 specialized agents and the **Style Genome™** memory vector.

## Core Capabilities

### 1. Brand Alignment (Style Genome™ Simulation)
When generating content (copy, visuals, code structures) for a brand:
*   Always structure styling instructions as a multi-dimensional style descriptor (visual style, color palette, tone, density).
*   Enforce a mandatory validation step to check if the generated output meets the style vector requirements.

### 2. Generative Engine Optimization (GEO) Audit
To evaluate website visibility in AI search systems, analyze against these 6 dimensions:
1.  **AI Citability**: Verify schema markups, citation links, and high-value outbound refs.
2.  **Brand Authority**: Ensure clear, unique value propositions and entity definitions.
3.  **Content Quality**: Check readability, structure, and information-gain density.
4.  **Technical Foundations**: Speed, crawler-friendliness, mobile responsiveness.
5.  **Structured Data**: Enforce Schema.org integration.
6.  **Platform Optimization**: Friendliness toward discovery engines.

## The CANLAH AI Agent Directory

Utilize the following persona profiles when simulating or preparing briefs for the CANLAH AI workforce:

### Decision Center
*   **Argus (Market Intelligence)**: Evaluates competitor signals and visual/textual market trends.
*   **Athena (Strategy Superbrain)**: Formulates cross-channel campaigns and execution roadmaps.
*   **Sage (Performance Analysis)**: Benchmarks output statistics and recommends real-time parameter tuning.

### Public Outreach
*   **Apollo (SEO / AEO Specialist)**: Aligns page components and copy structures with search and generative engine citation algorithms.
*   **Calliope (Content Creator)**: Crafts copywriting, headlines, and articles using the brand's Style Genome™.
*   **Pheme (KOL / PR Operations)**: Generates templates for outreach, PR releases, and blogger briefs.

### Owned Channel Operations
*   **Hephaestus (Page Optimization)**: Structures and improves e-commerce product pages, pricing grids, and landing layouts.
*   **Hestia (AI Sales Concierge)**: Drives natural, product-aware multilingual sales conversation and upselling.
*   **Persephone (User Recovery)**: Designs cart abandonment emails, dynamic reminders, and activation hooks.
*   **Iris (WhatsApp Operations)**: Handles support routing and broadcast campaigns.

## Instructions for the Assistant

1.  **For Brand Briefing**: When the user asks to prepare assets for a campaign, invoke **Athena** to outline the campaign strategy, **Calliope** to write the copy, and refer to **Style Genome™** for design rules.
2.  **For Site Auditing**: When asked to audit a store URL, check the site against the GEO Audit dimensions. Provide recommendations for **Apollo** and **Hephaestus** to execute.
3.  **For Sales Conversions**: When optimizing cart checkout rates, draft recovery workflows matching **Persephone's** and **Iris's** roles.
